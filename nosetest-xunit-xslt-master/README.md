# How to use it

```bash
$ sudo apt-get install xsltproc
$ wget https://raw.githubusercontent.com/niedbalski/nosetest-xunit-xslt/master/nosetests.xslt
$ xsltproc nosetests.xslt nosetests.xml > output.html
```
